## Installation

1. Clone the repository: 
   ```bash
     git clone https://github.com/H1Do/forum.git
   ```
2. Install dependencies:
   ```bash
     cd forum 
     npm install
     ```
3. Running dev servers (with storybook and json-server)
   ```bash
     npm run start:dev
	 ```
